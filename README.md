# Manipulação do DOM - Atividade 2 🧩✨

Um repositório com exercícios práticos de Manipulação do DOM em JavaScript. Ideal para quem está começando com Front-End e quer praticar seleção de elementos, eventos, criação e alteração dinâmica do conteúdo da página. 🚀

## Conteúdo do projeto 📚
- Exercícios e demonstrações que mostram como:
  - Selecionar elementos (querySelector, getElementById) 🔎
  - Manipular atributos, classes e estilos 🎨
  - Criar, inserir e remover nós do DOM ➕➖
  - Lidar com eventos (click, submit, input) 🖱️⌨️
  - Validar formulários simples ✅

## Como abrir / rodar 🖥️
1. Clone o repositório:
   ```bash
   git clone https://github.com/leandroFilipy/FE-manipulacao-DOM-atv2.git
   ```
2. Abra o arquivo `index.html` no seu navegador (duplo clique ou via Live Server no VS Code) 🌐
3. Experimente os exercícios e altere o código para ver o DOM reagir em tempo real! 🔁

## Estrutura do projeto 📁
- index.html — página principal com exemplos e exercícios 🧾  
- css/ — estilos do projeto 🎨  
- js/ — scripts com exemplos de manipulação do DOM 🧠  
- assets/ — imagens e recursos (se houver) 🖼️

## Tecnologias usadas 🛠️
- HTML5 🏷️
- CSS3 🎨
- JavaScript (DOM API) 📜

## Dicas para estudo 💡
- Leia a documentação da DOM API e teste pequenos trechos no console do navegador. 🧪  
- Faça alterações incrementais e verifique o que muda na página. 🔍  
- Crie pequenos projetos (todo list, contador, formulário dinâmico) para fixar os conceitos. 📝
